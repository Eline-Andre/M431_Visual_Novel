# Visual Novel, projet CFPT

Auteurs: Nhi, Eline, Lena

## Introduction
Le projet visait à faire un Visual Novel dans le cadre du module 431, au CFPT. Un visual novel (roman visuel) est un genre de jeu interactif, où l'on suit une histoire via du texte tout au long du jeu. Vient s'ajouter au texte un interface graphique (personnages, fond, etc).
